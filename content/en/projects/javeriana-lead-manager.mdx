## Overview

Built as a technical assessment for the Marketing Office of Pontificia Universidad Javeriana, this SPA lets a marketing team browse academic program offerings, capture and manage prospective-student leads, and check activity metrics — without a backend, using `localStorage` for persistence.

## What it includes

Program listings pulled from a REST API and rendered as filterable cards (by name and by category — undergraduate, graduate, continuing education); a lead-capture form with field validation (including an `@javeriana.edu.co` email hint) and input normalization; a metrics dashboard (total leads, same-day registrations, most-requested program, category breakdown); CSV export (UTF-8 BOM, Excel-compatible); and a manual dark-mode toggle with system-preference detection.

## Stack

React 19 with Vite, TypeScript in strict mode, Tailwind CSS, Context API for state, Framer Motion for animation, and Vitest for unit tests.

## What this project demonstrates

Turning a scoped technical assessment into a complete, polished SPA under real constraints — including test coverage and a working deployed demo, not just a code sample.
