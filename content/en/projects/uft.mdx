## Overview

**UFT** is a full-stack platform designed to **manage, score, and track Ultimate Frisbee tournaments in real-time**.

It combines **live statistics**, **tournament administration**, and **fan-facing experiences** into a single ecosystem used by clubs, leagues, and coaches.

---

## The Problem

Ultimate Frisbee tournaments often rely on:
- Manual scorekeeping
- Delayed statistics
- Fragmented tools (spreadsheets, WhatsApp, PDFs)

This leads to **poor data quality**, **slow reporting**, and **limited fan engagement**.

---

## The Solution

UFT provides:
- 🟢 **Real-time score tracking**
- 📊 **Live player & team statistics**
- 🏆 **Tournament management dashboard**
- 📱 **Mobile app for spectators (in progress)**
- 📸 **Integrated media & event coverage**

---

## Technical Architecture

### Frontend
- **React 18** for admin and public dashboards
- **React Native + TypeScript** for the mobile spectator app
- Tailwind CSS + shadcn/ui

### Backend
- **NestJS** as main backend framework
- RESTful API
- PostgreSQL as database

### Infrastructure
- Dockerized services
- Nginx reverse proxy
- Deployed on VPS (DigitalOcean)
- SSL + environment isolation
- Manual & automated backups
