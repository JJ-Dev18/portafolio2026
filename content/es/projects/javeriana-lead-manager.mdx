## Overview

Construida como prueba técnica para la Dirección de Mercadeo de la Pontificia Universidad Javeriana, esta SPA permite a un equipo de mercadeo explorar la oferta de programas académicos, capturar y gestionar leads de posibles estudiantes, y consultar métricas de actividad — sin backend, usando `localStorage` para persistencia.

## Qué incluye

Listado de programas consumido desde una API REST y presentado en cards filtrables (por nombre y por categoría — pregrado, posgrado, educación continua); un formulario de captura de leads con validaciones de campo (incluyendo un hint para email `@javeriana.edu.co`) y normalización de datos; un dashboard de métricas (total de leads, registros del día, programa más solicitado, distribución por categoría); exportación a CSV (UTF-8 BOM, compatible con Excel); y un toggle de modo oscuro manual con detección de preferencia del sistema.

## Stack

React 19 con Vite, TypeScript en modo estricto, Tailwind CSS, Context API para estado, Framer Motion para animaciones, y Vitest para tests unitarios.

## Qué demuestra este proyecto

Convertir una prueba técnica acotada en una SPA completa y pulida bajo restricciones reales — incluyendo cobertura de tests y una demo desplegada funcionando, no solo una muestra de código.
