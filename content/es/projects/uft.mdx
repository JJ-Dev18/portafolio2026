## Overview

**UFT** es una plataforma full-stack diseñada para **gestionar, puntuar y rastrear torneos de Ultimate Frisbee en tiempo real**.

Combina **estadísticas en vivo**, **administración de torneos** y **experiencias para fanáticos** en un único ecosistema utilizado por clubes, ligas y entrenadores.

---

## El Problema

Los torneos de Ultimate Frisbee a menudo dependen de:
- Registro manual de puntajes
- Estadísticas retrasadas
- Herramientas fragmentadas (hojas de cálculo, WhatsApp, PDFs)

Esto conduce a **datos de baja calidad**, **reportes lentos** y **participación limitada de los fanáticos**.

---

## La Solución

UFT proporciona:
- 🟢 **Seguimiento de puntajes en tiempo real**
- 📊 **Estadísticas en vivo de jugadores y equipos**
- 🏆 **Panel de gestión de torneos**
- 📱 **Aplicación móvil para espectadores (en progreso)**
- 📸 **Cobertura integrada de medios y eventos**

---

## Arquitectura Técnica

### Frontend
- **React 18** para paneles administrativos y públicos
- **React Native + TypeScript** para la aplicación móvil de espectadores
- Tailwind CSS + shadcn/ui

### Backend
- **NestJS** como framework principal del backend
- API RESTful
- PostgreSQL como base de datos

### Infraestructura
- Servicios dockerizados
- Nginx como reverse proxy
- Desplegado en VPS (DigitalOcean)
- SSL + aislamiento de entornos
- Respaldos manuales y automatizados
